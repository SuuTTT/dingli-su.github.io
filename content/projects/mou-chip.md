## Project Overview

The **Mou-Chip** architecture explores how **Task and Motion Planning (TAMP)** can benefit from learning-based abstractions. 

### Key Features
- **Hierarchical Decision Making:** Leveraging structural information principles for efficient planning.
- **Learning Abstractions:** Reducing state-space complexity for high-dimensional robotic tasks.
